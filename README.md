# calculus

Flashcards for Single-Variable Calculus — for spaced repetition review via [flashcards](https://github.com/thomasrribeiro/flashcards).

## Coverage

| File | Topic |
|------|-------|
| 01_limits_continuity | Limits, one-sided/infinite limits, $\epsilon$-$\delta$, squeeze theorem, continuity, IVT |
| 02_the_derivative | Derivative definition, tangent line, differentiability, higher-order derivatives |
| 03_differentiation_rules | Power, product, quotient, chain rule |
| 04_transcendental_derivatives | Trig, exponential, logarithmic, inverse trig derivatives; logarithmic differentiation |
| 05_implicit_related_rates | Implicit differentiation, related rates |
| 06_mvt_lhopital | Extreme value theorem, Rolle's, Mean Value Theorem, L'Hôpital's rule |
| 07_curve_sketching | Monotonicity, concavity, inflection, asymptotes, full sketching procedure |
| 08_optimization | Closed-interval method, applied optimization, Newton's method |
| 09_antiderivatives | Indefinite integrals, basic formulas, initial value problems |
| 10_definite_integrals_ftc | Riemann sums, definite integral properties, Fundamental Theorem of Calculus |
| 11_integration_techniques | $u$-sub, integration by parts, trig integrals, trig substitution, partial fractions |
| 12_applications_of_integration | Area, volume (disk/washer/shell), arc length, work, average value |
| 13_improper_integrals | Type 1/2 improper integrals, $p$-tests, comparison tests |
| 14_sequences_series_taylor | Convergence tests, power series, Taylor/Maclaurin series |
